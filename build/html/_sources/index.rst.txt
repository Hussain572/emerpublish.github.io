Welcome to the Roxar RMS documentation!
========================================

Roxar API builds bridges between different applications and extends the capability of internal products. It provides an easy-to-use approach for creating scripts, plugins, and standalone applications using the Python language. 

Some of the tasks that you can accomplish with **Roxar API** are as follows:

   * Retrieve data
   * Make a copy of data and store it into another data object
   * Run calculations against data 
   * Create new data items
   * Save projects as a new name

For a hands-on demo, refer to the :ref:`roxar`.
